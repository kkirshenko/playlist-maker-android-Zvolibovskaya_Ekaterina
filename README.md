# playlist-maker-android-Zvolibovskaya_Ekaterina

### Спринт 3
Проект реализует вёрстку главного экрана по макету с демо-уведомлениями и добавляет полноценную навигацию на экраны настроек и поиска.

### Спринт 4
Проект включает экраны поиска и настроек, свёрстанные по макету Figma, с навигацией на базе Compose Navigation.
Маршруты типизированы через enum, а переходы централизованно управляются компонентом PlaylistHost с использованием NavController.

### Спринт 5
Структура проекта переведена под чистую архитектуру.

### Спринт 6	
Создан SearchViewModel и SearchScreen.

### Спринт 7
Приложение отображает главный экран с FloatingActionButton, открывающим ModalBottomSheet; панель автоматически скрывается при нажатии на затемнённую область за её пределами.

### Спринт 8
Приложение реализует навигацию по разделам треков, плейлистов и избранного, поддерживая поиск композиций с индикатором загрузки, детальный просмотр треков и создание новых плейлистов с сохранением данных.
