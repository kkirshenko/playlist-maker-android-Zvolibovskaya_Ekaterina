# Playlist Maker

# playlist-maker-android-Zvolibovskaya_Ekaterina

**Playlist Maker** — это Android-приложение, которое позволяет искать музыкальные треки, создавать и управлять плейлистами. 

Приложение позволяет:

- Поиск треков;
- Создание и управление плейлистами;
- Добавление треков в избранное;
- Переключать светлую / тёмную тему;
- Хранить пользовательские настройки через DataStore.

---

## Технологии

- **Язык:** Kotlin;
- **UI:** Jetpack Compose (Material / Material3);
- **Архитектура:** Clean Architecture + MVVM;
- **DI:** Koin;
- **База данных:** Room;
- **Хранение настроек:** DataStore;
- **Асинхронность:** Coroutines + Flow;
- **Загрузка изображений:** Coil;
- **Сеть:** Retrofit.

---

## Сборка и запуск
- Склонировать репозиторий: git clone <repository-url>;
- Открыть проект в Android Studio;
- Дождаться завершения Gradle Sync;
- Убедиться, что выбран эмулятор или подключено Android-устройство;
- Нажать Run.

---

## Особенности реализации
- Тема приложения сохраняется между запусками с помощью DataStore;
- Данные плейлистов и треков хранятся в Room;
- Состояния экрана реализованы через StateFlow;
- UI автоматически реагирует на изменения данных.
